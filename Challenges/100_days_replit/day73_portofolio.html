Note taking app with login

  1. Fungsi get_info_account() untuk mendapatkan informasi username dan password dari pengguna.
  2. Fungsi hash_login() untuk menghasilkan hash dari password yang diberikan dengan salt tertentu.
  3. Fungsi add_user() untuk menambahkan pengguna baru ke database dengan password yang di-hash dan salt yang dihasilkan secara acak.
  4. Fungsi auth_challange() untuk melakukan autentikasi pengguna dengan memeriksa apakah username dan password yang diberikan cocok dengan yang tersimpan.
  5. Fungsi login() untuk menginisiasi proses login dengan memanggil auth_challenge().
  6. Fungsi load_note() untuk memuat catatan dari file jika ada, jika tidak, membuat daftar kosong.
  7. Fungsi add_note() untuk menambahkan catatan baru dengan timestamp.
  8. Fungsi view_note() untuk menampilkan semua catatan beserta timestamp-nya.
  9. Fungsi note_app() untuk menyediakan menu bagi pengguna untuk menambah atau melihat catatan.
  10. Fungsi main() sebagai fungsi utama yang mengontrol alur program dengan meminta pengguna untuk login atau mendaftar.

Simple Novel using Tkinter

Program ini menggunakan library Tkinter untuk membuat aplikasi GUI sederhana yang merupakan cerita interaktif. Berikut adalah fungsionalitas utamanya:

  1. Fungsi story1() dan story2() menampilkan gambar dan tombol untuk memilih jalannya cerita.
  2. Fungsi happy_ending() dan bad_ending() menampilkan gambar akhir cerita dan tombol untuk mengulang cerita.
  3. Fungsi home() menampilkan gambar awal dan tombol untuk memulai cerita dari awal.
  4. Program dimulai dengan memunculkan gambar awal dan dua tombol untuk memilih cerita.
  5. Saat tombol ditekan, fungsi yang sesuai dipanggil untuk menampilkan gambar dan tombol baru sesuai dengan pilihan pengguna.
